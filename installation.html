count map pulse breathe

from an installation at the joseph nease gallery 12/2019 - 2/2020

this work wove together sound, image, data, text, fiber and code. for the exhibit long fabric scrolls were suspended from the ceiling. a robotic quilting machine was programmed to create "maps" across the fabric surfaces with stitched counterpoint from my hand. this installation extended some of the alogorithmic processes that i explored in swarm. sound and animations were generated live by a chorus of computing machines.
